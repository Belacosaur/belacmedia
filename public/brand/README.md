# Belac Media — brand kit (static files)

This folder ships with the site under **`/brand/*`** (served from `public/brand/`).

| File | Purpose |
|------|---------|
| **`tokens.json`** | Machine-readable colours, type, logo paths, voice notes — import into design tools or codegen. |
| **`belac-media-brand-kit.md`** | Full written guidelines for designers and partners (usage, clear space, voice). |

**Interactive guide:** open **[Brand kit](/brand)** on the live site for swatches, logo grid, and copy-paste CSS variable names.
